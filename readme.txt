Bon dia, benvingut a la activitat avaluable 3 de Programació de Servicis i procesos (PSP).
En aquest video vaig a ficar a prova la aplicació que gestiona peticions HTTP.
He treballat en Spring Boot i he requerit del PostMan per a realitzar les peticions.

Este projecte consta de tres clases: ClaseController, FiltreExtensio i el Main.

Espere que la aplicació complisca amb els requeriments de l'activitat avaluable.

Tant el codi com la documentació estarà disponible tant en el github com en la entrega de florida oberta. :)

PD: en la entrega està adjuntat el pom.xml, la versio de java utilitzada es la '20.0.2'
